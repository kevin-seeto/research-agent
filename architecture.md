# Architecture Decision Record

## Search Layer
Decision: Tavily API as primary, DuckDuckGo as fallback
Reason: Tavily provides deeper content for AI agents.
DuckDuckGo ensures the agent never fails due to API limits.

## Reasoning Layer
Decision: Manual ReAct loop over LangChain AgentExecutor
Reason: More transparent, easier to debug, more reliable
across LangChain version changes.

## Memory Layer
Decision: JSON file persistence over in-memory storage
Reason: Survives reboots and scheduled runs. Simple to
inspect and debug. Sufficient for single-user daily runs.

## Email Layer
Decision: Raw SMTP over MIME libraries
Reason: MIME encoding caused ASCII codec failures with
special characters generated by the LLM. Raw SMTP with
explicit ASCII encoding is more reliable.

## Cost Decision
Decision: Claude Haiku as default LLM
Reason: Sufficient reasoning capability at ~$0.01 per run.
Sonnet available as upgrade for deeper analysis.
