Privacy Policy

Last updated: June 2026

Overview

Personal Research Agent is a private automation tool that searches the web and delivers research briefings via email. This privacy policy explains how the application handles data.

Data Collection

This application does not collect, store, or share any personal data from third parties. The only data processed is:

  • Research topics specified by the user in the .env file
  • Web search results from DuckDuckGo and Tavily APIs
  • Email credentials stored locally in the .env file
  • Run history stored locally in memory.json

Data Storage

All data is stored locally on the user's machine. No data is transmitted to any server other than:

  • Anthropic API - for LLM reasoning (research topic and search results only)
  • Tavily API - for web search queries
  • DuckDuckGo - for web search queries
  • Gmail SMTP - for email delivery

Third Party Services

This application uses the following third party services:

  • Anthropic Claude API - Privacy Policy
  • Tavily API - Privacy Policy
  • DuckDuckGo - Privacy Policy
  • Google Gmail - Privacy Policy
  • LinkedIn API - Privacy Policy

Credentials

API keys and email credentials are stored locally in a .env file on the user's machine. These credentials are never committed to GitHub or shared with any third party.
